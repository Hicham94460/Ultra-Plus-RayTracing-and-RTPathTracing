1. Go to https://www.techpowerup.com/download/nvidia-dlss-dll/

2. Download nvngx_dlss.dll

3. (Download nvngx_dlssg.dll from https://www.techpowerup.com/download/nvidia-dlss-3-frame-generation-dll/
   if you use frame gen)

4. Download nvngx_dlssd.dll if you want to - the latest version from Alan Wake (20 November 2023) works fine
   https://www.techpowerup.com/download/nvidia-dlss-3-ray-reconstruction-dll/

4. Copy these files to Cyberpunk 2077\bin\x64 (I rename the old files to *.original first)


***IMPORTANT: For any of the files you replace:
- Right click > Properties
- Place a checkmark next to "Unblock"
- Click OK
